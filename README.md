Суровая многопользовательская арена по мотивам классического [Bomberman](https://github.com/ChikageBB/bomberman-game). Никакого рандома и усилителей — только равные условия, бомбы, лабиринты из блоков и 3 минуты на то, чтобы доказать свое превосходство над соперниками!

## ✨ Особенности геймплея
- 👥 **Мультиплеер на 4 игроков:** Собирайтесь с друзьями и сражайтесь на одной арене.
- ⏱️ **Матчи по 3 минуты:** Короткие, но предельно интенсивные раунды. Динамика не проседает ни на секунду.
- 🏆 **Система очков:** Побеждает не обязательно тот, кто выжил последним. Зарабатывайте очки, разрушая блоки лабиринта и подрывая других игроков. У кого больше очков к концу таймера — тот и победитель!

## 🛠 Технологии
- **Язык программирования:** Java
- **Графический интерфейс:** Java Swing
- **Отрисовка и графика:** Graphics2D (AWT)
